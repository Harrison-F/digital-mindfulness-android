# Digital Mindfulness Android

Native Android/GrapheneOS APK for counting selected website visits by local calendar day.

- Install/update path: GitHub Releases via Obtainium
- Current MVP: local-only visit counts using Android Accessibility Service signals
- Future sync: open-source self-hostable backend, documented but intentionally deferred

See `docs/android-apk-obtainium-plan.md` for the architecture and Obtainium flow.
